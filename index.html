BONJOUR ET SOYEZ LA BIENVENUE!


A l'ère du développement numérique, la plupart des entreprises et marques possèdent un site Web pour faciliter leur communication. Mais au-delà du simple rôle de canal de communication, les sites Web se révèlent comme un outil vital de développement de la notoriété d'une entreprise et de vente.
Un site Web, encore appelé site internet ou simplement site, est un assemblage de pages Web ordonnées qui représente généralement l'image d'une personne physique ou morale (entreprise, startup, association, freelancer, etc.) sur internet. Ces différentes pages sont interconnectées entre elles par des liens encore appelés URL (Uniform Resource Locator) dont l'accès est défini par une adresse web. Il existe plusieurs moyens de création d'un site internet dont le plus connu est l'utilisation du langage Html, CSS ou WordPress. Les sites web sont généralement hébergés sur un serveur web et sont accessibles via plusieurs canaux dont généralement les moteurs de recherche comme Google, Bing, etc.

Pendant le cours de Web design, j'ai appris à créer des pages Web avec les langages Html, CSS et l'utilisation de Bootstrap. A cet effet, j'ai eu l'opportunité de créer mon site web personnel composé des différentes pages web crées tout au long du cours de web design. Je vous laisse le découvrir.
